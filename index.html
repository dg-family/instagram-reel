instagram-reel

若你看到本頁，代表 GitHub Pages 已成功發佈靜態內容（倉庫根目錄的 index.html）。

說明：本專案的主程式是 FastAPI（Python），需在伺服器上執行；GitHub Pages 只能放靜態網頁，無法在這個網址上跑轉錄／下載 Instagram 的 API。

請到原始碼倉庫查看說明與本機／雲端部署方式：

github.com/dg-family/instagram-reel

本機快速啟動（需已安裝依賴與 ffmpeg、yt-dlp）：

  • pip install -r requirements.txt
  • python web_server.py → 瀏覽 http://127.0.0.1:8765/
